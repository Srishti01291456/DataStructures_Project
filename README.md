# DataStructures_Project
This is a repository where there will be dsa based projects from beginner to advanced level.
A Data Structures-based Student Record Management System designed to efficiently manage student data using core concepts like Linked Lists, Stacks, Queues, and Sorting Algorithms.

The system supports operations such as adding, deleting, searching, and updating student records, along with dynamic data handling and optimized retrieval. Advanced features include sorting student records by marks using efficient algorithms (Merge Sort/Quick Sort), stack-based undo functionality, and queue-based processing for record management.

This project demonstrates strong understanding of DSA fundamentals, memory management, and algorithm optimization, while also focusing on clean code structure and modular design.

Tech Stack: Java, OOP, Data Structures & Algorithms
